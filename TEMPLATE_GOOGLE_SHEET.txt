📝 EXPLICACIÓN DE CADA COLUMNA:
================================

┌─────────────────────────────────────────────────────────────────┐
│ 1. ID (Número)                                                  │
│    • Número único de cada mueble (1, 2, 3, 4...)              │
│    • NO REPITAS números                                         │
│    • Se genera automáticamente si usas el panel admin          │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│ 2. Nombre (Texto)                                               │
│    • Nombre descriptivo del mueble                             │
│    • Ejemplo: "Sofá Moderno de 3 Plazas"                       │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│ 3. Categoria (Texto) ⚠️ IMPORTANTE                             │
│    • Debe ser EXACTAMENTE una de estas opciones:               │
│      - Sala                                                     │
│      - Comedor                                                  │
│      - Dormitorio                                               │
│      - Oficina                                                  │
│      - Almacenamiento                                           │
│    • Respeta mayúsculas y acentos                              │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│ 4. Precio (Número)                                              │
│    • Solo números y punto decimal                              │
│    • NO uses símbolos como $, comas, etc.                      │
│    • Ejemplo correcto: 899.99                                   │
│    • Ejemplo incorrecto: $899,99                                │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│ 5. Descripcion (Texto)                                          │
│    • Descripción completa del mueble                            │
│    • Incluye materiales, medidas, características              │
│    • Puedes escribir varias líneas                              │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│ 6. Imagenes (Texto) ⚠️ IMPORTANTE                              │
│    • Nombres de archivos separados por COMAS                   │
│    • Ejemplo: foto1.jpg, foto2.jpg, foto3.jpg                  │
│    • Las fotos deben estar en la carpeta /imagenes             │
│    • NO incluyas "imagenes/" en el nombre                       │
│    • Mínimo 1 imagen OBLIGATORIA                                │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│ 7. Videos (Texto) - OPCIONAL                                    │
│    • Nombres de archivos de video separados por COMAS          │
│    • Ejemplo: video1.mp4, video2.mp4                            │
│    • Los videos deben estar en la carpeta /videos              │
│    • NO incluyas "videos/" en el nombre                         │
│    • Deja vacío si no hay videos                                │
└─────────────────────────────────────────────────────────────────┘


✏️ CÓMO AGREGAR UN NUEVO MUEBLE:
=================================

1. Ve a la última fila con datos
2. Crea una nueva fila abajo
3. Llena cada columna siguiendo el formato de arriba
4. Guarda (automático en Google Sheets)
5. Recarga tu página web
6. ¡Listo! El mueble aparecerá automáticamente

❌ ERRORES COMUNES:
==================

ERROR: Precio con símbolo $
❌ Precio: $899.99
✅ Precio: 899.99

ERROR: Categoría mal escrita
❌ Categoria: sala (minúscula)
❌ Categoria: SALA (mayúscula completa)
✅ Categoria: Sala (primera letra mayúscula)

ERROR: Imágenes con ruta completa
❌ Imagenes: imagenes/sofa.jpg
✅ Imagenes: sofa.jpg

ERROR: Múltiples imágenes sin comas
❌ Imagenes: foto1.jpg foto2.jpg foto3.jpg
✅ Imagenes: foto1.jpg, foto2.jpg, foto3.jpg


🎨 TIPS AVANZADOS:
==================

• Puedes usar URLs completas para imágenes:
  Imagenes: https://ejemplo.com/foto.jpg

• Puedes mezclar archivos locales y URLs:
  Imagenes: sofa.jpg, https://ejemplo.com/sofa2.jpg

• El orden de las imágenes importa: la primera será la principal

• Deja celdas vacías si no tienes videos, ¡no pasa nada!

• Puedes ocultar filas que no quieras mostrar (clic derecho > ocultar)
